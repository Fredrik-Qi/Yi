# 如何在 Mac 上打包成本地应用

您现在可以通过以下简单的三步，将这个项目变成一个真正的 Mac 桌面应用（.dmg）：

### 第一步：导出代码
点击 AI Studio 界面右上角的 **"Share"** 按钮旁边的箭头（或设置菜单），选择 **"Export to ZIP"**。将解压后的文件夹放在您的 Mac 桌面。

### 第二步：安装环境
打开您的 Mac **终端 (Terminal)**，进入该项目目录，运行：
```bash
npm install
```

### 第三步：生成 App (.dmg)
在终端运行：
```bash
npm run electron:build
```

### 结果
打包完成后，您会在项目根目录下看到一个 `release` 文件夹，里面就有 `YiMind.dmg`。您可以直接把它拖进 **应用程序 (Applications)** 文件夹中。

---

### 技术特性总结：
1. **安全性**: 代码运行在 Electron 提供的沙箱环境中，保持 100% 本地运行，不连接任何外部数据库。
2. **字体**: 已应用您选择的 **龙藏体 (Long Cang)**，在 Mac 视网膜屏幕上会有非常好的呈现。
3. **交互**: 采用了 Apple 风格的 `hiddenInset` 标题栏，整体界面如同 macOS 原生组件。
